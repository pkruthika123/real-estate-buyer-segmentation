# real-estate-buyer-segmentation
Machine Learning Based Buyer Segmentation and Investment Profiling for Real Estate Market Intelligence
a# Real Estate Buyer Segmentation

## Project Overview
This project applies machine learning clustering techniques to segment real estate buyers based on demographics, investment behavior, and financing patterns.

## Objectives
- Identify different types of buyers
- Analyze investment patterns
- Provide business recommendations

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib / Seaborn

## Models Used
- K-Means Clustering
- Hierarchical Clustering

## Features
- Buyer segmentation
- Investment profiling
- Interactive dashboard

## Dataset
- clients.csv
- properties.csv

## Results
The model identified key buyer segments such as:
- First-Time Buyers
- Global Investors
- Corporate Buyers
- Luxury Investors
