# Real Estate Buyer Segmentation

## Project Title
Machine Learning Based Buyer Segmentation and Investment Profiling for Real Estate Market Intelligence

## Project Overview
This project uses machine learning clustering techniques to segment real estate buyers based on demographics, acquisition purpose, loan behavior, geography, and investment patterns.

## Objectives
- Identify different buyer groups
- Analyze investment behavior
- Understand financing patterns
- Provide business recommendations
- Build an interactive dashboard

## Datasets Used
- clients.csv
- properties.csv

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## Machine Learning Models
- K-Means Clustering
- Hierarchical Clustering

## Project Workflow
1. Data cleaning
2. Feature engineering
3. Dataset merging
4. Exploratory data analysis
5. Encoding and scaling
6. Clustering
7. Cluster profiling
8. Business insights
9. Streamlit dashboard

## Output
The project identifies meaningful buyer segments such as:
- First-Time Buyers
- Global Investors
- Corporate Buyers
- Luxury Investors
